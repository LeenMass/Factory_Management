# Factory Management System - Frontend

A modern web application designed to streamline factory operations, allowing efficient management of employees, departments, and shifts, and tracking each user's daily activity count.

## Technologies Used

- JavaScript  
- React.js  
- Axios (API requests)  
- Git / GitHub (version control)  
- CSS

## Installation

Follow these steps to set up the project locally:

```bash
# 1. Clone the repository
git clone https://github.com/LeenMass/Factory_Management_frontend.git

# 2. Navigate to the project folder
cd Factory

# 3. Install dependencies
npm install

# 4. Run the development server
npm run dev

The frontend should now be running at http://localhost:3000.
```
