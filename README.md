# Factory Management System Frontend

A web-based frontend application for the **Factory Management System**, designed to handle employees, departments, and shift assignments.  
Includes features for employee management, department categorization, and secure access using **JWT authentication**.

## Technologies Used

- JavaScript  
- React.js  
- Axios for API requests  
- Git / GitHub for version control  
- CSS

## Installation

1. **Clone the repository**
```bash
git clone https://github.com/LeenMass/Factory_Management_frontend.git
2. **Navigate to the project folder** : cd Factory
3. **Install dependencies** : npm i
4. **Run the development server** : npm start

Backend API

This frontend connects to the backend API repository here:
https://github.com/LeenMass/Factory_Management_api

Make sure the backend server is running to use all features.
